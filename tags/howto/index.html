Installing Darwin 8.0.1 as a VM Copper is an installation of Darwin 8.0.1. Installing Darwin is…finicky. Avoid installing to ufs, that seems to work very poorly. ESXi configuration Virtual Machine - version 8 Hardware Hard disk 1 should be attached to IDE bus (0:0) 768MB of RAM seems to be plenty. E1000 ethernet adapter works here as well.
